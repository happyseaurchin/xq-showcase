# XQ Showcase

Interactive mathematical visualizations exploring psycho-social dynamics.

## About

This site showcases artifacts created through XQ mathematics explorations - visualizations of Riemann spheres, roots of unity, pscale dynamics, and more.

## Stack

- **Frontend**: React + TypeScript + Vite
- **Database**: Supabase (awareness-functions)
- **Deploy**: Vercel

## Local Development

```bash
npm install
npm run dev
```

Create `.env.local`:
```
VITE_SUPABASE_URL=https://ovknmrdvgcozigffevza.supabase.co
VITE_SUPABASE_ANON_KEY=your-anon-key
```
